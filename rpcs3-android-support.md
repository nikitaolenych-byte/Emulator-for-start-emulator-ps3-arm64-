# Підтримка RPCS3 ARM64 Windows на Android

## Де отримати допомогу
- Офіційний сайт RPCS3: https://rpcs3.net/
- Форум Termux: https://github.com/termux/termux-packages/discussions
- Issues цього репозиторію

## Часті питання
- Чому не запускається RPCS3? — Перевірте, чи встановлені всі залежності, чи є підтримка ARM64.
- Як додати BIOS/ігри? — Скопіюйте файли у директорію емулятора.
- Чи працює графічний інтерфейс? — Потрібен X сервер на Android.
