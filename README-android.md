# Запуск RPCS3 ARM64 Windows через Termux на Android

## Вимоги
- Android ARM64
- Termux
- box64
- wine

## Інструкція

1. Встановіть Termux з Google Play або F-Droid.
2. Скопіюйте скрипт start-rpcs3-android.sh у домашню директорію Termux або репозиторій.
3. Дайте права на виконання:
   chmod +x start-rpcs3-android.sh
4. Запустіть скрипт:
   ./start-rpcs3-android.sh
5. Дочекайтесь встановлення box64, wine та завантаження RPCS3.
6. Емулятор запуститься через Wine.

## Примітки
- Посилання на RPCS3 ARM64 Windows у скрипті потрібно замінити на актуальне.
- Для коректної роботи потрібен x11-repo у Termux.
- Продуктивність залежить від потужності пристрою.

---

### Додатково
- Якщо виникнуть проблеми з запуском, перевірте лог Wine та box64.
- Для запуску ігор потрібні BIOS та образи ігор PlayStation 3.
